# 案例 02 · 中秋節社群活動設計 Brief · 90 分鐘流程拆給 90 秒

> **角色:** 設計師(也適合 PM 自己想 brief 給設計師)
> **場景:** 客戶突然要中秋節社群活動,3 天內提案
> **預計耗時:** 首次 25 分鐘 · 熟練後 **3 分鐘**
> **預期產出:**
> - [x] 結構化設計 Brief(目標 / 客群 / 禁忌 / 必含元素 / 尺寸清單)
> - [x] 3 個視覺方向草案(色票 + 構圖概念 + 字體 + 參考方向)
> - [x] FB / IG Story / LINE 三尺寸的素材適配清單

---

## 步驟 1 · 把 PM 給的爛 brief 變結構化(8 分鐘)

### 場景

PM 在 LINE 給你這段:

```
今天客戶說中秋節要做社群活動,客戶 XX 食品(老牌餅廠),
品牌色橘黃色,預算 5 萬,3 天內要,設計給 FB IG LINE 用。
```

少了:目標客群、視覺禁忌、有沒有 logo 需求、上線後跟誰追蹤、字級規範。

### 動作

按 **⌘3** 進設計協作助手,貼 PM 的 LINE 訊息 + 這句:

```
PM 給我這段 brief。幫我列 5 個必須反問 PM 的關鍵問題,
按優先序(沒問就無法開工的先)。台灣老牌客戶語境。
```

### AI 應該回你

```
反問 PM 5 題(優先序):

1. 目標客群年齡層:25-35 / 35-45 / 45+
   → 影響色彩飽和度、字體現代度、文案語氣
2. 客戶舊包裝有沒有強制復用元素(吉祥物 / 標準字)
   → 老牌客戶常有不可動的 brand asset
3. 預算 5 萬包不包含 KOL / 廣告投放?
   → 5 萬若全做素材設計剛好 · 若含投放只夠 3 張圖
4. 中秋連假是 9/X 起 · 活動需要在連假前還是中發?
   → 決定第 1 張上線時機
5. CTA 是什麼:導購買、抽獎、純品牌曝光?
   → 影響是否需要設計按鈕、QR Code 區塊
```

### 你做的事

把這 5 題複製,轉貼回 PM。**一次問完不來回 10 次。**

---

## 步驟 2 · PM 答完後 · 產 3 個視覺方向(10 分鐘)

### PM 回答

```
1. 25-35 歲女性
2. 沒有強制吉祥物 · 但 logo 必含
3. 5 萬只含設計 · 不含投放
4. 連假前 1 週開始發
5. CTA 是電商導購
```

### 動作

繼續同一個對話(助手會記得上一步),貼:

```
PM 答完 5 題了:[貼 PM 答案]

幫我想 3 個視覺方向。每個方向給我:
1. 名稱(一句話 · 中文)
2. 主色 3 色(hex)+ 一句話色彩說明
3. 構圖概念(視覺中心 / 視覺路徑)
4. 適合的字體家族(中文 + 英文)
5. 參考方向關鍵字 3 組(我自己去 Pinterest / Behance 搜)
6. 文案 hook 1 句(上 FB 標題用)
```

### AI 應該回你 3 組類似這樣

```
方向 A · 月光家宴
- 主色:#F4A460 落日橘 / #4A6B3A 墨綠 / #F5E6D3 米白(暖光圓潤)
- 構圖:上 1/3 滿月雲彩、下 2/3 家宴俯視長桌、品牌 logo 右下
- 字體:思源宋體 + Cormorant Garamond(古典典雅)
- 參考關鍵字:harvest moon illustration / vintage family dinner / Asian mid-autumn
- Hook:「月圓人圓 · 一口傳承三代滋味」

方向 B · 摩登月見
- 主色:#FF8B47 亮橘 / #1A2B4A 深夜藍 / #FFFFFF 留白(對比強烈)
- 構圖:抽象幾何月、品牌 logo 大方居中、產品平面化擺放
- 字體:思源黑體 + Inter(極簡現代)
- 參考關鍵字:minimal mid-autumn poster / contemporary moon design / flat lay
- Hook:「中秋 · 用最簡單的方式 · 表達不簡單的心意」

方向 C · 剪紙窗花
- 主色:#E64A19 朱紅 / #F5C84C 蜜黃 / #2D2D2D 墨黑(傳統民藝)
- 構圖:左 1/3 剪紙月兔、中 1/3 產品展示、右 1/3 窗格紋裝飾
- 字體:王漢宗中楷 + DIN(民俗 × 現代)
- 參考關鍵字:Chinese paper cutting / mid-autumn traditional / craft poster
- Hook:「老味道 · 新祝福」
```

### 你做的事

選 1 個方向(或混搭)· 開 Figma / Illustrator 實際畫。**之前要花 90 分鐘想方向 · 現在 10 分鐘有 3 組可選。**

---

## 步驟 3 · 多渠道素材適配清單(5 分鐘)

社群活動 = FB / IG Story / LINE 至少 3 尺寸 · 之前手動裁圖花 30 分鐘。

### 動作

```
我選方向 A 月光家宴。FB 1200x1200 已經畫好(資訊在中央)。
幫我列 IG Story 9:16、LINE 圖 3:4、報紙半版 28:21 三個尺寸的適配清單。

每個尺寸標註:
- 完整尺寸(px)
- 主視覺要怎麼裁(保留哪部分)
- 文字位置(避開系統 UI 遮擋區)
- 留白規則
```

### AI 給的清單(類似這樣)

```
IG Story · 1080x1920 (9:16)
- 主視覺:上 1/3 月亮雲彩(裁掉左右雲彩細節)
- 中 1/2:長桌變成局部特寫(只保中央 3 個杯子)
- 文字:標題放下 1/4 · 避開底部 250px(IG Story 工具列)
- Hook 字級 60pt 以上 · 確保手機可讀

LINE 圖 · 1040x1040(LINE OA 廣播)
- 主視覺維持 FB 構圖等比縮小
- 標題改放右下角(LINE 預覽會被「點圖看內文」遮上方)

報紙半版 · 26.5x18cm
- 構圖完全重畫(直版改橫版)
- 主視覺改月亮 + 家宴 1:2 比例
- 字級 72pt+ 標題 · 24pt 副標 · 14pt 內文
```

### 你做的事

照清單做,不會做完才發現 IG Story 字被切。

---

## 步驟 4 · 文案附 hashtag(2 分鐘 · 加碼)

社群文案不是你的事,但你常常被 PM 半夜問「這張圖配什麼字」。

```
方向 A 月光家宴 FB 貼文文案 · 80 字內 · 加 5 個 hashtag。
語氣:溫暖懷舊但不老氣。CTA 導購買官網。
```

AI 給你完整貼文 · 直接貼給 PM。

---

## 完成後

- [ ] 3 個方向 doc 給 PM 提案用
- [ ] 1 份結構化 brief(reverse 自反問 PM 的答案)
- [ ] 多尺寸適配清單
- [ ] 給 👍 或 👎 · 助手會學

**省時對比:**
- 之前:來回 LINE 問 PM 1 小時 + 想方向 90 分鐘 + 多尺寸來回 30 分鐘 = **3 小時**
- 現在:整個流程 **25 分鐘**

---

## AI 偏題時 · 5 句話救

1. **太抽象** → 「給我具體 hex 色票 · 不要『暖色系』這種字」
2. **太通用** → 「客戶是台灣老牌餅廠 · 不是新創品牌 · 重來」
3. **太多元素** → 「每個方向只挑 3 個視覺重點 · 多了會雜」
4. **沒考慮預算** → 「5 萬只夠靜態圖 · 不要建議影片或動圖」
5. **講理論** → 「不要解釋色彩學 · 直接給我可執行的方向」
