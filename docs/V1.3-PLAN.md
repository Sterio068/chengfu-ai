| # | 標題 | 來源 | 估時 |
|---|---|---|---|

| C2 | PDPA dry_run audit 顯示在 /admin/audit-log 過濾器 | R29-R32 累積 dry_run 紀錄會塞 audit · 需可篩 | 0.3d |

### C1 詳解 · Real Mongo 測

當前 160 tests 全用 mongomock · 但 R30/R31 證明 mongomock 跟真 Mongo 行為差:
- arrayFilters($[n]) → mongomock NotImplementedError · prod 走 atomic
- TTL index 行為 → mongomock 不會自動 expire
- collation case-insensitive → mongomock 部分支援

v1.3 加 `tests/integration/` 跑真 Mongo container · CI 跑 docker compose:
- pytest --integration 跑 docker compose up mongo
- 測 PDPA arrayFilter prod path
- 測 TTL index 真會 expire(用 sleep + count)
- 測 case-insensitive prod regex
