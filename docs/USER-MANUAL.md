## 第一次登入

### 步驟 1 · 取得帳號
管理員會給你:
- email(公司 email)
- 預設密碼(請首次登入後改)
- 系統網址 `http://智慧助理.local` 或 Cloudflare Tunnel 連結

### 步驟 2 · 登入
1. 開瀏覽器(建議 Chrome / Safari)
2. 進系統網址
3. 輸入 email + 密碼 · 按「登入」

### 步驟 3 · 改密碼(強烈建議)
1. 右上角頭像 → 個人設定
2. 找「密碼」· 改成只有你知道的

### 步驟 4 · 看 Onboarding 4 步教學
首次登入會跑 4 步引導,3 分鐘看完。
之後想再看:右上「?」→「重看新人引導」

## 6 個 Workflow 閉環

進「下一步建議」(workflows) 看完整清單。

每個 workflow 有兩種用法:
1. **拆下一步草稿**:AI 產步驟給你檢查再送出(穩)
2. **🚀 直接執行**:主管家自動串接所有專家一次跑完(每日上限 5 次)

### 1️⃣ 投標完整閉環
招標 PDF → 9 欄結構 → Go/No-Go → 預算 → 建議書大綱

### 2️⃣ 活動完整企劃
活動主題 → 場景 Brief + 主視覺 → 預算分配

### 3️⃣ 新聞發布閉環
事實 → 新聞稿 → 媒體邀請 Email

### 4️⃣ 結案完整閉環(v1.67)
專案資料 → KPI 報表 + NPS 問卷 + 內部復盤 → CRM 紀錄

### 5️⃣ 月底營運閉環(v1.67 · admin 用)
月損益 + 標案漏斗 → 老闆月報 Email

### 6️⃣ 客戶提案完整閉環(v1.67)
客戶 brief → 創意 3 方向 → 報價 3 檔 → PPT 大綱 8 章

## 常見 Q&A

### Q1 · AI 怎麼說錯話了 / 編造內容
**A**:每個 agent 看到敏感事項會標「需人工確認」· 你必須自己確認再送出。
若 AI 答錯了 · 點 👎 + 寫備註 · 系統會學習(v1.75 學習 loop)。

### Q2 · 機敏資料(客戶選情 / 內部財務)能丟給 AI 嗎?
**A**:看資料分級(海報貼牆):
- L1 公開 · 可直接丟雲端
- L2 一般 · 去識別化後可丟
- L3 機敏 · **絕不**上雲 · 走本地處理(階段二 Ollama)

主管家會自動偵測 L3 · 提醒你。

### Q3 · 對話可以保存多久?
**A**:對話 / 上傳檔案 100% 存在公司 Mac mini 本地 · 不上雲。
保留 365 天 · 之後自動清(PDPA)。

### Q4 · 我用太多會被限制嗎?
**A**:輕度 200 萬 / 中度 150 萬 / 重度 300 萬 token / 月。
超過 80% 系統 email 提醒 · 100% 自動暫停。中控可看自己用量。

### Q5 · 主管家「直接執行」上限多少?
**A**:每天 5 次。系統設計防失控。
如果該天用完 · 改用「拆下一步草稿」模式(無上限)。

### Q6 · 我能訓練 AI 學承公司風格嗎?
**A**:可以 · 兩個方式:
- 上傳優質範本到知識庫(資料 → 上傳)· AI 自動引用
- 給好的回答按 👍 · 系統學習偏好

### Q7 · 找不到舊對話怎麼辦?
**A**:對話內按 🕒 歷史對話 · 看最近 10 筆。
更舊的請用 ⌘K 搜尋對話內容。

### Q8 · 同事問我「主管家怎麼這麼會」 · 我不知道怎麼解釋
**A**:給他看這份手冊的「6 個 workflow 閉環」。
重點:主管家不是單一 AI · 它是「AI 經理」· 會把任務分派給 9 個專家。

## 疑難排解

### 系統打不開 / 載入很慢
1. 按 ⌘⇧R 強制重新整理
2. 清瀏覽器 cache(Chrome:設定 → 隱私 → 清除瀏覽資料)
3. 仍無法 → Slack 告訴 admin · 或 LINE Sterio

### 對話沒回應 / 卡住
1. 重新整理(⌘⇧R)
2. 看右下「系統正常」是綠燈嗎?紅燈表示後端有問題
3. 換另一個 agent 試試
4. 仍無 → admin 檢查 Mac mini 是否還在跑

### 上傳檔案失敗
- 單檔 < 25 MB(超過要壓縮 / 拆檔)
- 一次最多 6 個附件
- 支援格式:PDF / Word / Excel / PPT / TXT / MD / JSON / 圖片(PNG/JPG/WebP/GIF)

### 對話框太小看不清楚
按右上 ⛶ 全螢幕 · 或拖左緣調寬

### 看不到中控 / 商機 / 會計
這些是 admin only · 你不是 admin 看不到。聯絡 admin 給你權限。

### 系統升版後行為不一樣
看 release note(管理員會通知)· 或問 admin。
