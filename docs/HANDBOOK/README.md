# 企業 AI 手冊 · 依角色閱讀

> 不同角色看不同手冊。3-5 分鐘讀完自己那份就能用。

## 單一角色

| 你是 | 讀這份 | 讀多久 |
|---|---|---|
| 老闆 / 管理階層 | [01-老闆快速上手](./01-BOSS.md) | 5 分鐘 |
| PM / 專案經理 | [02-PM 三件事](./02-PM.md) | 10 分鐘 |
| 設計師 / 美編 | [03-設計師兩件事](./03-DESIGNER.md) | 5 分鐘 |
| 業務 / 行銷 | [04-業務一件事](./04-SALES.md) | 5 分鐘 |

## 混合角色(本公司 10 人公司常見)

| 你身兼... | 先看哪 2 頁 | 為什麼 |
|---|---|---|
| **PM + 業務** | [02-PM](./02-PM.md) → [04-業務](./04-SALES.md) | 投標流程是主軸 · 標案監測是業務面 |
| **PM + 設計** | [02-PM](./02-PM.md) → [03-設計師](./03-DESIGNER.md) | 你是 PM 兼設計總監 · 要看「跟設計師溝通」+「Brief 結構化」 |
| **設計 + 行銷** | [03-設計師](./03-DESIGNER.md) → [04-業務](./04-SALES.md) | 設計交付完接著要寫社群文案 · 兩本互補 |
| **老闆 + PM**(小公司) | [01-老闆](./01-BOSS.md) → [02-PM](./02-PM.md) | 看儀表也要會用 · 才知道同仁在說什麼 |
| **任何 + AI Champion** | 自己角色 + [QUICKSTART](../QUICKSTART.md) + 4 份角色全讀 | 你要當公司教練 · 全套都熟 |
