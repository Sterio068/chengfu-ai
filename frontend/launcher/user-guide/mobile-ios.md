# 📱 iPhone / iPad 設定完整指南

> 場勘 PWA / audio note / 知識庫拍照 都用 iPhone · 必先設好 4 件

---

## ❶ 相機格式 → 改「最相容」(必)

**為什麼**:iPhone 預設拍 HEIC · Claude Vision 不認 · 上傳會 400 錯
**怎麼**:
1. iPhone 設定 app
2. 相機
3. 格式
4. 選 **「最相容」**(預設是「高效率」)

之後拍的照片都是 JPEG。**舊照仍是 HEIC**(可在照片 app 個別 → 分享 → 另存 JPEG)

---

## ❷ 麥克風權限(audio_note 必)

**為什麼**:場勘錄 30 秒語音 + Whisper 轉文字 · iOS 預設拒網頁取麥克風
**怎麼**:
1. 設定 → Safari → 麥克風 → **「詢問」或「允許」**
2. 第一次開 launcher 場勘 view 點 🎙 → iOS 跳對話 → 允許

**已拒過怎麼救**:
- 設定 → Safari → 進階 → 網站資料 → 找 launcher 域名 → 移除
- 重新開啟 launcher · 重新請求權限

---

## ❸ GPS 定位(場勘必)

**為什麼**:拍照 + 自動帶 GPS · 之後 admin 看地圖
**怎麼**:
1. 設定 → 隱私權與安全性
2. 定位服務 → **開啟**(總開關)
3. 滑下找 Safari · 選「使用 App 期間」
4. 「精確位置」**開**(否則只到街區級)

**地下室 / 室內訊號弱**:GPS 抓不到 · launcher 會問你手動填地址(不擋上傳)

---

## ❹ 相簿存取(部分功能)

**為什麼**:從相簿選舊照(不是現拍)· iOS 會問
**怎麼**:設定 → Safari → 照片 → 「選取的照片」或「全部」

---

## 🚀 安裝 PWA 到主畫面(可選 · 像 native app)

1. Safari 開 `https://ai.<company_ai>.com/`
2. 下方分享 icon → **加入主畫面**
3. icon 取「智慧助理」
4. 之後從主畫面開 · 不顯示瀏覽器 chrome · 體感像 app

---

## ✅ 4 設定一次檢查 checklist

- [ ] 相機 → 格式 → 最相容(JPEG)
- [ ] Safari → 麥克風 → 詢問/允許
- [ ] 隱私權 → 定位 → Safari → 使用 App 期間
- [ ] Safari → 照片 → 選取的/全部
- [ ] 加 launcher 到主畫面(可選)

---

## 常見卡關

### Q1 · 拍完上傳 400 「HEIC 不支援」
→ 設定 ❶ 沒做 · 改「最相容」· 重拍

### Q2 · 點 🎙 沒反應
→ 設定 ❷ · Safari 麥克風被拒 · 移除網站資料重來

### Q3 · GPS 一直「取得中...」轉圈
→ 設定 ❸ · 確認 Safari「使用 App 期間」+「精確位置」開
→ 室內試用「手動填地址」flow 替代

### Q4 · 「相簿無法存取」
→ 設定 ❹ · iOS 14+ 預設「選取的照片」· 改「全部」
